Censys:
curl -H "Authorization: Bearer <censys_api>" -H "Accept: application/vnd.censys.api.v3.host.v1+json" https://api.platform.censys.io/v3/global/asset/host/<ip>



Shodan: 
curl -s -o response.json -w "%{http_code}" "https://api.shodan.io/shodan/host/search?key=<Shodan_api_key>&query=org:%22<name of org>%22"
type response.json
